# gen_pass

Простой bash-скрипт для генерации случайных паролей.

## Использование

Скрипт принимает один параметр — длину пароля.

Пример:

    ./gen_pass.sh 10

Вывод:

    пароль: hG82kdP0xA

## Установка

Склонировать репозиторий:

    git clone https://github.com/qwemiracle1337/genpass.git
    cd genpass

Сделать файл исполняемым:

    chmod +x gen_pass.sh
